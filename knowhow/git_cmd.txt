gitでよく使うコマンド一覧

・ローカルにリモートリポジトリ(master)をクローンする
$ git clone git@github.com:suzuki05/suburi.git

・リモートリポジトリをoriginに登録、以降originと打つだけで良い(originとはリモートリポジトリ(master)である)
$ git remote add origin git@github.com:suzuki05/suburi.git

・リモートリポジトリのURLを確認
$ git remote -v

・ローカル環境のファイルをgitで管理するときの初期設定
$ git init

・ブランチごとの履歴を見る
$ git log 201912151130 --oneline

・リモートリポジトリ(master)からローカルのmasterに変更をpullし、margeする
$ git checkout master
$ git pull origin master
$ git checkout newbranch

・pullの内訳(fetchではリモート追跡ブランチ(origin/master)のことをわかっている必要がある。)
$ git fetch origin master (リモートリポジトリoriginのmasterブランチをリモート追跡ブランチ(origin/master)に落とす)
$ git merge origin/master (ローカルリポジトリ内のリモート追跡ブランチmasterよりローカルのmasterブランチにマージする。)

・commit履歴を見る
$ git log newbranch --oneline

・ローカルで管理しているブランチの表示
$ git branch

・ブランチ作成
$ git branch newbranch

・ブランチ移動
$ git checkout newbranch

・ローカルにあるブランチを削除
$ git branch -D oldbranch

・変更のあったファイルを確認
$ git status

・変更をadd
$ git add .

・addしたファイルをcommit
$ git commit -m "message"

・ブランチの変更をリモートリポジトリ(master)に反映させる PR
$ git push origin newbranch
